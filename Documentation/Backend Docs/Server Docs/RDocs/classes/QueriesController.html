create()

Processes an array of query requests in JSON format sent as a POST request to /queries/create. The results of each query is pushed to an array that is rendered in JSON format. Each response includes the response of the query as well as a human-readable name for the associated query that can easily be rendered on the front end. See Bunker Motor Warriors Query API documentation for more details.
